UMassOnline API Instructions

POST URL

API calls are made to the following URL, depending on the target environment:

Environment POST URL
Test https://{your.test.hostname}/webapps/umol-umol-api-BBLEARN/
Production https://{your.prod.hostname}/webapps/umol-umol-api-BBLEARN/

POST Parameters

Name Description Required
username Supplied by UMOL Yes
auth Supplied by UMOL Yes
action Method name, listed below Yes
id Identifier Yes

Possible Actions (choose one)

Method Name Returns ID Parameter Basis
GetUserByUsername User details Username
GetCoursesByUsername User and their course memberships Username
GetCourseByCourseID Course details Course ID
GetGradesByCourseID Grades for course members with student role Course ID

Sample API Request

Example: Get the details for a course with the ID of "ART-205":

POST https://mycourses.umassonline.net/webapps/umol-umol-api-BBLEARN/ username=test auth=1234 action=getCourseByCourseID id=ART-205

Sample API Response

The API will return the following XML for the request above:

<results> <course> <batch_id>ART-205</batch_id> <course_id>ART-205</course_id> <description>ART 205 Fall 2013</description> <is_available>true</is_available> <is_child>false</is_child> <is_parent>false</is_parent> <title>ART 205 Fall 2013 Session</title> </course> </results>
